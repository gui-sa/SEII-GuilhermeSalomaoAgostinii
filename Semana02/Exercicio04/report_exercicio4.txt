Sobre a referência bibliográfica Python in 100 seconds (https://www.youtube.com/watch?v=x7X9w_GIm1s):
    - A linguagem de programação mais popular (2021)
    - linguagem amigável e com boa curva de aprendizado
    - Comunidade forte
    - Tipagem possibilita facilidade de entendimento e DEBUG, facilitando a manutenção
    - Surgiu em 1991
    - Usado para criar aplicações do lado do Servidor (Server Side)
        - WEB APPS (django)
        - Machine Learning
        - Estatística e manipulação do BIG DATA
    - Não é fortemente tipada
    - Aceita POO (e é uma linguagem pautada em POO)
    - Possui Package Manager (PIP).
